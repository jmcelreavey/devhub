# PWA

DevHub can be installed as a Progressive Web App from supported browsers.

## What This Gives You

- App-like launcher icon.
- Standalone browser window.
- Faster access from your desktop or mobile device.

## Install

Open DevHub in a browser that supports PWA installation, then use the browser's install option.

The exact menu name varies by browser.

## Mobile Use

DevHub can work well on mobile when it is reachable from your local network.

Only use LAN access on networks you trust. DevHub does not include built-in authentication.

## Service Worker

DevHub includes a service worker for app-like behavior. It should not be treated as an offline-first data sync system.

If something looks stale, reload the page or restart the dashboard.
