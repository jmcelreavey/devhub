# Project Setup

**Use when:** new project, scaffold, bootstrap, greenfield, from scratch, starter

When scaffolding a new project:

- Choose boring technology unless the requirements demand otherwise
- Set up linting, formatting, and pre-commit hooks from day one
- Create a `.env.example` with all required variables documented
- Include a basic CI pipeline (even if it only runs tests and lint)
- Write a README with: what it does, how to run it, how to test it
